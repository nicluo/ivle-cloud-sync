How to move or rename your Cloud Sync app folder

Overview

Cloud Sync uses a standard app folder (i.e. all files are in [Dropbox Folder]/Apps/IVLE Cloud Sync/). Deleting this folder on any system might cause the application to unlink, revoking Cloud Sync access, sort of like an "uninstall".

To move or rename the IVLE Cloud Sync folder, you have to log in to the Dropbox website, and modify the folders there. We have also included a picture guide for the most technologically disinclined users to follow!

1 Renaming the app folder

  1. Go to https://www.dropbox.com/home/Apps, the default location for the IVLE Cloud Sync folder.
  2. Right click "IVLE Cloud Sync", then select "Rename".
  3. Choose whichever name you fancy.

2 Moving the app folder

  1. Go to https://www.dropbox.com/home/Apps, the default location for the IVLE Cloud Sync folder.
  2. Right click your IVLE Cloud Sync folder ("IVLE Cloud Sync" by default), then select "Move".
  3. Choose whichever new location you prefer.

Note that in the example the new app folder is at [Dropbox Folder]/IVLE, convenient indeed!
